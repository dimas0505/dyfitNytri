# DyFit Nytri Diet Prescription App

## Project Overview
DyFit Nytri is a diet prescription application designed to help users manage their dietary needs effectively. The app provides personalized meal plans, nutritional information, and user-friendly interfaces to track dietary habits and progress.

## Setup Instructions

### Prerequisites
- [Node.js](https://nodejs.org/) (version x.x.x)
- [npm](https://www.npmjs.com/) (version x.x.x)
- A modern web browser (Chrome, Firefox, etc.)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dimas0505/dyfitNytri.git
   ```
2. Navigate into the project directory:
   ```bash
   cd dyfitNytri
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the App
1. Start the application:
   ```bash
   npm start
   ```
2. Open your web browser and go to `http://localhost:3000` to view the app.
